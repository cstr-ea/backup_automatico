# backup_automatico
cript em Python para automação de backup de arquivos, desenvolvido com foco em práticas básicas de infraestrutura. O projeto realiza a cópia completa de diretórios, preservando a estrutura original e organizando os backups em pastas nomeadas com data e hora, evitando sobrescrita de dados.

A solução foi projetada para ser simples, eficiente e facilmente adaptável a diferentes ambientes, podendo ser integrada a rotinas automatizadas de backup em sistemas Windows ou Linux.
